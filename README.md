# URL Shortener with Analytics

A full-stack web application that generates shortened URLs and tracks click analytics with a React frontend and Flask backend. Demonstrates RESTful API design, database integration, and production cloud deployment.

## Features
* RESTful API for URL shortening and analytics
* React frontend with real-time click tracking dashboard
* PostgreSQL database for persistent storage
* Cloud deployment on Render (backend) and Vercel (frontend)
* Production-ready with Gunicorn WSGI server
* CORS-enabled cross-origin requests

## Tech Stack
Python | Flask | PostgreSQL | React | Gunicorn | Render | Vercel

## Live Demo

* **Backend API:** https://url-shortener-main.onrender.com
